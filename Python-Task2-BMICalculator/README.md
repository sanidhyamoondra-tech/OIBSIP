# OIBSIP
# Basic BMI Calculator

A simple, interactive command-line Python application that calculates a user's Body Mass Index (BMI) and categorizes their health status. This project is a task for the Oasis Infobyte Python Programming internship.

## Features
* **BMI Calculation:** Computes BMI using the standard metric formula (weight / height^2).
* **Health Classification:** Categorizes the calculated result into standard brackets: Underweight (<18.5), Normal (18.5-24.9), Overweight (25-29.9), and Obese (≥ 30).
* **Input Validation:** Automatically rejects non-numeric inputs and negative numbers with user-friendly error messages to prevent crashes.
* **Continuous Loop:** Prompts the user to perform multiple calculations in a single session.

## Tech Stack
* **Language:** Python 3.x
* **Components:** Built-in Python functions. No external libraries or dependencies are required.

## Getting Started

### Prerequisites
Ensure you have Python 3 installed on your machine. You can verify this by running `python --version` in your terminal or command prompt.

### Installation & Execution
1. Download the project files or clone the repository.
2. Open your terminal or command prompt to run the code.
